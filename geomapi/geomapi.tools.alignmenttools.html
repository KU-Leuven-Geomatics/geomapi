Determines the close ennoug sessions based on bounding volumes

Args:

boundingVolume (np.array 6x1): The test bounding volume referenceSessions (list[SessionNode]): All the sessionNodes to check

Returns:

list[SessionNode]: All the close enough session nodes

Calculates the weighted average of a list of PoseEstimations

poses (PoseEstimation): The list of Pose Estimations

Returns:

np.array 4x4: The Average transformation matrix

testSession (SessionNode): The Test Session where the positions needs to be determined from refSessions (list[SessionNode]): The Reference sessions to help calculate the new position

Returns:

np.array: The estimated transformation
