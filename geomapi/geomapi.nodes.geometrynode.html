geomapi.nodes.geometrynode

GeometryNode is an abstract Python geometry class to govern the data and metadata of geometric data (Mesh, BIM, PointClouds). It is the base class for the MeshNode, PointCloudNode and BIMNode classes. It contains the base concepts to manipulate cartesianBounds, orientedBounds, orientedBoundingBoxes, etc.

IMPORTANT: the GeometryNode class is an archytype class from which specific data classes (e.g. PointCloudNode) inherit. Do not use this class directly if you can use a child class with more functionality.

__init__(graph: rdflib.Graph = None, graphPath: str = None, subject: rdflib.URIRef = None, path: str = None, cartesianBounds: array = None, orientedBounds: array = None, orientedBoundingBox: open3d.geometry.OrientedBoundingBox = None, **kwargs)

Creates a GeometryNode from one or more of the following inputs. If successful, it will set the following attributes.

Features:

  1. cartesianBounds(np.array [6x1]): [xMin,xMax,yMin,yMax,zMin,zMax]

  2. orientedBounds (np.ndarray [8x3]): bounding points of OrientedBoundingBox

  3. orientedBoundingBox (o3d.geometry.OrientedBoundingBox)

Args:

1.graph (Graph, optional): An RDF Graph to parse.

2.graphPath (str, optional): The path of an RDF Graph to parse. If no subject is provided, the first subject of the graph is retained.

3.path(str,optional): A filepath to a resource.

4.subject (URIRef, optional): A subject to use as identifier for the Node. If a graph is also present, the subject should be part of the graph.

5.name (str, optional): A name of the Node. This is not a unique identifier but serves as non-functional description.
